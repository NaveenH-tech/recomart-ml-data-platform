## 3. Problem Statement

RecoMart is an e-commerce company that wants to build a scalable recommendation system. The objective is to create an end-to-end data management pipeline capable of ingesting, validating, processing, storing, and transforming data before training recommendation models.

---

## 4. Objectives

- Build an automated data pipeline
- Validate and clean data
- Engineer recommendation features
- Store reusable features
- Track datasets and experiments
- Train recommendation models
- Orchestrate the complete workflow
